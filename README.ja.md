# 🔫 **ガニファイ**

**ガニファイ** は、マーカーをベースにした WebAR 体験で、銃のマーカーを認識し、ブラウザ上でインタラクティブな3D武器モデルと詳細な銃器情報を即座に表示します。

![Gunify](https://github.com/KrishBharadwaj5678/Gunify/raw/main/GunifyDemo.jpg)

## 🚀 **機能**

| 機能                     | 説明                               |
| ---------------------- | -------------------------------- |
| 🎯 **マーカーベースARトラッキング** | AR.js + A-Frame を使用してマーカーを検出します。 |
| 🔫 **武器情報パネル**         | タイトル、説明、スペックを即座に表示します。           |
| 🔊 **リアルな銃声音**         | 各銃に**発砲音**と**リロード音**を搭載しています。    |
| 🌐 **あらゆるデバイスで動作**     | アプリのインストール不要で、ブラウザ上でそのまま動作します。   |

## 📦 **インストール & セットアップ**

### 1️⃣ リポジトリをクローン

```bash
git clone https://github.com/KrishBharadwaj5678/Gunify.git
```

### 2️⃣ フォルダへ移動

```bash
cd Gunify
```

### 3️⃣ ローカルサーバーで実行

* **Live Server** をインストール
* `index.html` を右クリック → **Live Serverで開く**

---

## 💡 使い方

1. リポジトリ内の *images* フォルダから **ARターゲット画像** をダウンロードします。
2. ブラウザでページを開きます。
3. カメラを画像に向けると、3Dモデルが表示されます。
