# 🔫 **Оружифай**

**Оружифай** — это веб-AR-проект на основе маркеров, который распознаёт маркеры оружия и мгновенно отображает интерактивные 3D-модели оружия с подробной информацией прямо в браузере.

![Оружифай](https://github.com/KrishBharadwaj5678/Gunify/raw/main/GunifyDemo.jpg)

## 🚀 **Функции**

| Функция                         | Описание                                           |
| ------------------------------ | -------------------------------------------------- |
| 🎯 **Маркерное AR-отслеживание** | Распознаёт маркеры оружия с помощью AR.js + A-Frame. |
| 🔫 **Информационная панель оружия** | Показывает название, описание и характеристики мгновенно. |
| 🔊 **Реалистичные звуки оружия** | Включает звуки **выстрела** и **перезарядки** для каждого оружия. |
| 🌐 **Работает на любом устройстве** | Без установки приложений — работает прямо в браузере. |

---

## 🛠️ **Технологический стек**

| Технология             | Назначение                                 |
| ---------------------- | ------------------------------------------ |
| 🟩 **AR.js + A-Frame** | Отслеживание маркеров и AR-рендеринг       |
| 🎨 **HTML / CSS**      | Разметка интерфейса + стеклянный дизайн     |
| ⚙️ **JavaScript**      | Логика, обработка событий, динамическая информация об оружии |

## 📦 **Установка и запуск**

### 1️⃣ Клонировать репозиторий

```bash
git clone https://github.com/KrishBharadwaj5678/Gunify.git
````

### 2️⃣ Перейти в папку проекта

```bash
cd Gunify
```

### 3️⃣ Запустить через локальный сервер

* Установите **Live Server**
* Нажмите правой кнопкой на `index.html` → **Открыть с помощью Live Server**

---

## 💡 **Использование**

1. Скачайте **AR-изображения-метки** из папки *images* в репозитории.
2. Откройте страницу в браузере.
3. Наведите камеру устройства на изображение, и появится 3D-модель.
