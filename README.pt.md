# 🔫 **Gunify**

**Gunify** é uma experiência WebAR baseada em marcadores que reconhece marcadores de armas e renderiza instantaneamente modelos 3D interativos de armas com informações detalhadas diretamente no navegador.

![Gunify](https://github.com/KrishBharadwaj5678/Gunify/raw/main/GunifyDemo.jpg)

## 🚀 **Recursos**

| Recurso                                      | Descrição                                                                        |
| -------------------------------------------- | -------------------------------------------------------------------------------- |
| 🎯 **Rastreamento AR baseado em marcadores** | Detecta marcadores de armas usando AR.js + A-Frame.                              |
| 🔫 **Painel de informações da arma**         | Exibe título, descrição e especificações instantaneamente.                       |
| 🔊 **Sons realistas de armas**               | Inclui sons de **tiro** e **recarregamento** para cada arma.                     |
| 🌐 **Funciona em qualquer dispositivo**      | Sem necessidade de instalação de aplicativo — funciona diretamente no navegador. |

---

## 🛠️ **Tecnologias utilizadas**

| Tecnologia             | Finalidade                                                      |
| ---------------------- | --------------------------------------------------------------- |
| 🟩 **AR.js + A-Frame** | Rastreamento de marcadores e renderização AR                    |
| 🎨 **HTML / CSS**      | Layout da interface + estilo glassmorphism                      |
| ⚙️ **JavaScript**      | Lógica, manipulação de eventos, informações dinâmicas das armas |

## 📦 **Instalação e configuração**

### 1️⃣ Clonar o repositório

```bash
git clone https://github.com/KrishBharadwaj5678/Gunify.git
```

### 2️⃣ Entrar na pasta

```bash
cd Gunify
```

### 3️⃣ Executar usando um servidor local

* Instale o **Live Server**
* Clique com o botão direito em `index.html` → **Abrir com Live Server**

---

## 💡 Uso

1. Baixe as **imagens de alvo AR** da pasta *images* do repositório.
2. Abra a página no seu navegador.
3. Aponte a câmera do seu dispositivo para a imagem e veja o modelo 3D aparecer.
